# EXPENSE TRACKER
Expense Tracker Web Application. Manage, Visualise and track all your Expenses in one place.
<br>
*Built With*
- **React**
- **Sass**


## To run this project

```javascript
   npm install  //To install dependencies
   npm start    // To launch it
```
