# EXPENSE TRACKER
Expense Tracker Web Application with React JS and SaSS. Manages, Visualises and Helps tracks all your Expenses.
*Built With*
- **React**
- **Sass**


## To run this project

```javascript
   npm install  //To install dependencies
   npm start    // To launch it
```
